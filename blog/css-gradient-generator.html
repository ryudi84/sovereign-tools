CSS Gradient Generator: Create Beautiful Gradients

Published Feb 22, 2026 · 5 min read

Flat colors are boring. Gradients add depth, energy, and personality to your designs — and CSS makes them surprisingly powerful. This guide covers everything from basic linear gradients to advanced conic patterns, with copy-paste code and trending palettes for 2026.

Linear Gradients

The most common type. A smooth transition between colors along a straight line.

background: linear-gradient(135deg, #667eea, #764ba2);

Direction Options

  • to right — left to right (same as 90deg)
  • to bottom — top to bottom (default, 180deg)
  • to bottom right — diagonal
  • 135deg — precise angle control

Multiple Color Stops

background: linear-gradient(
  90deg,
  #ff6b6b 0%,
  #feca57 33%,
  #48dbfb 66%,
  #ff9ff3 100%
);

Color stops let you control exactly where each color starts and ends. Without percentages, colors are distributed evenly.

Radial Gradients

Radiate outward from a center point — perfect for spotlight effects, buttons, and backgrounds.

background: radial-gradient(
  circle at 30% 50%,
  #6ee7b7,
  #0a0a0a
);

Shape and Position

  • circle vs ellipse (default) — controls the shape
  • at center, at top left, at 30% 70% — controls the origin
  • closest-side, farthest-corner — controls the size

Conic Gradients

Rotate around a center point like a color wheel. Great for pie charts, color pickers, and artistic effects.

background: conic-gradient(
  from 0deg,
  #ff6b6b,
  #feca57,
  #48dbfb,
  #ff9ff3,
  #ff6b6b
);

Note the last color matches the first — this creates a seamless rotation.

Advanced Techniques

Gradient Text

h1 {
  background: linear-gradient(135deg, #6ee7b7, #3b82f6);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}

Animated Gradients

@keyframes gradient-shift {
  0%   { background-position: 0% 50%; }
  50%  { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

.animated-bg {
  background: linear-gradient(-45deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
  background-size: 400% 400%;
  animation: gradient-shift 15s ease infinite;
}

Gradient Borders

.gradient-border {
  border: 2px solid transparent;
  background-image:
    linear-gradient(#0a0a0a, #0a0a0a),
    linear-gradient(135deg, #6ee7b7, #3b82f6);
  background-origin: border-box;
  background-clip: padding-box, border-box;
}

Mesh-Like Gradients

Layer multiple radial gradients for organic, mesh-style backgrounds:

background-color: #0a0a0a;
background-image:
  radial-gradient(at 40% 20%, rgba(110,231,183,.3) 0, transparent 50%),
  radial-gradient(at 80% 0%, rgba(59,130,246,.3) 0, transparent 50%),
  radial-gradient(at 0% 50%, rgba(192,132,252,.2) 0, transparent 50%);

Browser Support and Performance

CSS gradients have excellent support — 98%+ of browsers handle linear and radial gradients. Conic gradients are at 95%+. No vendor prefixes needed in 2026.

Performance tips:

  • Gradients are rendered by the GPU — they're fast and resolution-independent
  • Avoid animating background directly; animate background-position or use a pseudo-element
  • Complex gradients with 10+ stops can impact paint time on low-end devices
  • Use will-change: background-position for animated gradients
