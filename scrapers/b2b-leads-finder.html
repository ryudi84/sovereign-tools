API Quick Start

Integrate lead generation into your sales pipeline.

import { ApifyClient } from 'apify-client';

const client = new ApifyClient({ token: 'YOUR_API_TOKEN' });

const run = await client.actor('sovereigntaylor/b2b-leads-finder').call({
  searchQuery: 'CTO',
  industry: 'SaaS',
  location: 'San Francisco, CA',
  companySize: '51-200',
  maxResults: 100,
  includeEmails: true,
  includePhones: true
});

const { items } = await client.dataset(run.defaultDatasetId).listItems();
console.log(`Found ${items.length} leads`);
from apify_client import ApifyClient

client = ApifyClient("YOUR_API_TOKEN")

for lead in items:

Frequently Asked Questions

Emails are verified using MX record validation and SMTP checks. The tool discovers email patterns from company websites (e.g., first.last@company.com) and verifies that the mailbox exists. Each verified email is marked with emailVerified: true.

Typical email discovery rate is 60-80% of leads. Phone number discovery is lower, around 20-40%. Rates vary by industry and company size. Tech companies and larger firms tend to have higher discovery rates.

The tool aggregates publicly available data from company websites, team pages, about pages, and public professional directories. It does not access any private databases or violate any login-protected resources.

Yes. Provide a list of company domains (e.g., ["stripe.com", "notion.so"]) and the tool will scrape team/about pages directly to find contacts. This is faster and more accurate than keyword-based search.
